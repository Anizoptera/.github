# Anizoptera GitHub Profile

This repository controls the public GitHub organization profile.

## Rules

- Keep copy truthful, compact, and broad enough for the whole lab.
- Do not mention a “revival” narrative.
- Do not make support, package, or production-use claims without evidence.
- Keep legacy repository disposition out of the public profile.
- Install hooks with `just bootstrap` before committing.

## Commands

- `just bootstrap` — install tracked Git hooks and run checks.
- `just check` — validate profile files and shell scripts.
